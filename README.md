# Marketing Analytics & Customer Sentiment Analysis

End-to-end marketing analytics project combining **SQL, Python NLP, and Power BI** to analyse customer engagement, conversion rates, and review sentiment.

# Business problem

Fictious sportswear equipment retailer has little data analytical capability within the marketing team

Noticed decline in online engagement and erratic conversation rates from views to sales but needs an exploratory analysis to reveal trends and initial insights especially focusing on online engagement and variance through the year and by product

The project demonstrates a full analytics workflow:
- Data extraction from SQL Server
- Sentiment analysis using Python (NLTK VADER)
- Data transformation and enrichment
- Loading results back into SQL Server
- Interactive Power BI dashboard
- Executive-level insights presentation

---

# Project Architecture

SQL Server → Python ETL (Sentiment Analysis) → SQL Tables → Power BI Dashboard → Leadership Presentation

Pipeline stages:

1. Extract marketing and review data from SQL Server
2. Perform sentiment analysis on customer reviews
3. Categorise and bucket sentiment scores
4. Write enriched results back to SQL Server
5. Build Power BI visualisations and insights
6. Present key findings and recommendations
